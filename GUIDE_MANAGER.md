# Guide Utilisateur — Manager 👔
## Plateforme ICES — Gestion RH & Compétences

---

### 🌟 Introduction
La plateforme ICES vous permet de piloter votre équipe, de valider les demandes de vos collaborateurs et de suivre la performance et les compétences collectives.

---

### 🔐 Connexion & Premier accès
- **Comment se connecter** : Utilisez vos identifiants professionnels (email + mot de passe).
- **Interface** : Votre tableau de bord affiche les indicateurs clés de votre équipe (effectif, absences, taux de présence).

---

### ✅ Valider les demandes de congé
Approuvez ou refusez les demandes d'absence de vos collaborateurs.
1. Sur votre **Tableau de Bord**, consultez la section "Actions en attente".
2. Pour chaque demande, cliquez sur **Approuver** ou **Refuser**.
3. En cas de refus, un commentaire explicatif est obligatoire.
- **Résultat attendu** : Le collaborateur reçoit une notification immédiate de votre décision.

### 👥 Consulter le dossier de l'équipe
Accédez aux informations professionnelles de vos collaborateurs.
1. Allez dans la section **Mon Équipe**.
2. Cliquez sur **Voir profil** à côté du nom d'un collaborateur.
3. Vous y trouverez ses compétences, sa performance et son historique.
- 💡 **Astuce** : Utilisez cet espace pour préparer vos entretiens annuels.

### 📊 Suivre la productivité (Kanban)
Gérez les projets et les tâches assignés à votre équipe.
1. Allez dans la section **Projets**.
2. Déplacez les cartes entre les colonnes (À faire, En cours, Terminé).
3. Cliquez sur une carte pour voir les détails ou changer le responsable.
- **Résultat attendu** : Une vision claire de l'avancement des travaux en temps réel.

### 📈 Analyser les compétences (Gaps)
Identifiez les besoins en formation de votre équipe.
1. Allez dans la section **Compétences Équipe**.
2. Consultez le graphique radar pour voir les forces et faiblesses.
3. Identifiez les "Gaps" (écarts) entre les compétences actuelles et les besoins du poste.
- ⚠️ **Attention** : Ces données servent de base à l'élaboration du plan de formation par la RH.

### 📊 Pilotage de la Productivité (Ranking)
Suivez les performances individuelles et collectives.
1. Allez dans la section **Productivité Équipe**.
2. Consultez le **Ranking collectif** pour voir le classement des collaborateurs.
3. Surveillez les **Alertes de seuil** (en rouge) pour les collaborateurs sous l'objectif de 70%.
- 💡 **Astuce** : Utilisez ces indicateurs pour féliciter les meilleurs (Trophées/Médailles).

### 📅 Planning de Projet (Gantt)
Visualisez les jalons et les délais de vos projets.
1. Dans la section **Projets**, basculez sur la vue **Gantt**.
2. Suivez la timeline horizontale pour vérifier le respect des échéances.
- **Résultat attendu** : Une vue d'ensemble sur le semestre.

---

### ❓ Questions fréquentes
- **Puis-je modifier une décision de congé déjà prise ?** Non via l'interface, contactez la RH pour toute modification après validation.
- **Comment ajouter un nouveau collaborateur dans mon équipe ?** L'ajout est géré par la RH, contactez-les pour toute mise à jour d'effectif.
